# WireGuard Windows Installer

Companion Windows client installer for `wireguard-install.sh`.

The Linux script installs the WireGuard/wg-easy VPN server on Ubuntu, Debian,
RHEL-compatible systems, Amazon Linux, and cloud VMs. This PowerShell script is
for Windows client machines. It installs the official WireGuard Windows app and
can import a `.conf` tunnel profile generated by wg-easy.

## What It Does

- Installs WireGuard for Windows with `winget`
- Supports optional package version pinning
- Imports a `.conf` tunnel into the WireGuard manager service
- Optionally installs the tunnel as a Windows service
- Supports uninstall through `winget`

## Requirements

- Windows 10 1809 or newer, Windows 11, or supported Windows Server
- PowerShell 5.1 or newer
- Administrator PowerShell session
- `winget` / Microsoft App Installer
- A WireGuard client `.conf` file from wg-easy

## Install Only

```powershell
Invoke-WebRequest `
  -Uri "https://raw.githubusercontent.com/timmyzai/installers/main/wireguard-install-windows.ps1" `
  -OutFile "wireguard-install-windows.ps1"

Set-ExecutionPolicy -Scope Process Bypass -Force
.\wireguard-install-windows.ps1 -Action Install
```

## Install And Import Client Config

```powershell
.\wireguard-install-windows.ps1 `
  -Action InstallAndImport `
  -ConfigPath "C:\Users\you\Downloads\client.conf"
```

## Pin A Specific Package Version

```powershell
.\wireguard-install-windows.ps1 `
  -Action Install `
  -PackageVersion "1.0.1"
```

If `-PackageVersion` is omitted, `winget` installs the current package version
available from the Windows Package Manager source.

## Install Tunnel As A Windows Service

The default import path adds the config to the WireGuard manager so the user can
activate it from the WireGuard UI. For always-on service-style deployment:

```powershell
.\wireguard-install-windows.ps1 `
  -Action InstallAndImport `
  -ConfigPath "C:\Users\you\Downloads\client.conf" `
  -InstallTunnelService
```

## Replace Existing Tunnel

```powershell
.\wireguard-install-windows.ps1 `
  -Action Import `
  -ConfigPath "C:\Users\you\Downloads\client.conf" `
  -Force
```

## Uninstall

```powershell
.\wireguard-install-windows.ps1 -Action Uninstall
```

## Notes

- This script does not create a WireGuard server on Windows.
- Use `wireguard-install.sh` on the Linux server first.
- Keep `.conf` files private. They contain VPN client private keys.
- Review scripts before running them as Administrator.
